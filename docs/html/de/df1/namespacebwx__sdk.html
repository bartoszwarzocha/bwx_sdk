float BWX_EXPORT bwx_sdk::bwxIntermediate ( float a,
float b,
float x )

Returns a normalized value in the range [0, 1] based on x within the interval [a, b].

This function performs a smooth interpolation (using a smoothstep function) where the result is 0.0 if x is less than a, 1.0 if x is greater than or equal to b, and smoothly interpolates between 0 and 1 for values between a and b.

Parameters
aThe start of the interval.
bThe end of the interval.
xThe value to interpolate.
Returns
A normalized value between 0.0 and 1.0.
