◆ Init()

bool bwx_sdk::bwxInternat::Init ( const wxString & short_name = wxEmptyString)

Initializes the locale using a language short name.

If the provided short name is empty, the system language is used. If the system language is unknown, the default language is used instead. Once the language code is determined, the locale is initialized and the language catalogs are loaded.

Parameters
short_nameThe short name of the language to initialize.
Returns
True if initialization and catalog loading succeeded; false otherwise.
